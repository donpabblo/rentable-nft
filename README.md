# Install
```
npm i
```

# Start local hardhat node
```
npx hardhat clean
npx hardhat node
```

# Setup and Run
On a different terminal:
```
npx hardhat compile
npx hardhat deploy
npx hardhat mint --num 30
npx hardhat make-rentable --num 30
npm run build
npm run start
```
Go to https://localhost:3000
Enjoy!

# Test Contract
```
npx hardhat test
```
